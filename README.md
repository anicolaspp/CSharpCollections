# CSharpCollections

Some interesting `Collections` and `Data Structures` in C#.

This lib can be added from NuGet using 

```
PM> Install-Package DataCollections -Version 1.0.0
```

# Basic Structures

- Stack
- Queue
- Binomial Heap
- Circular Linked List
- Dictionary
- DisJoinSet
- Fibonacci Heap
- Skip List

# Graphs

- Adjacency List Graph
- Matrix Graph
